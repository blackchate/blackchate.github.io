Driveshare - Class Project

Car-sharing management service created for administrators, passengers and drivers.

Rides can be published and booked using the Android mobile application while records linked to the rides can be sorted, searched and managed using the Administrator web application (Android, Web). Implemented using Heroku, SpringBOOT, Vue.js and Android Studio.
