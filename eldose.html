ElDose - ConUHacksIII

Created at ConUHacksIII, Eldose is a web application made to monitor and manage medical dosages for clients.

SMS-based notifications are sent to users using the Twilio REST API reminding them to take their prescribed dosages at appropriate times.

Frontend: HTML, CSS, Bootstrap, JQuery, Ajax

Backend: PHP, SQL, Apacheserver
